Java 应用程序和Android应用程序子线程异常的不同结果

在普通的Java应用中，如果一个子线程执行时出现异常，那么只是该子线程退出。其他线程不受影响。
但是在Android应用里面，如果子线程执行时出现异常，则整个应用会退出。
虽然两者本质都是Java程序，但虚拟机的实现其实还是稍有差异的。

并行开发Java应用和Android应用的开发者需要注意。
